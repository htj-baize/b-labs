# B-Labs

B-Labs：由多个 AI Agent 组成的虚拟产品团队（含人类协作）。

- 团队手册：[`TEAM_MANUAL.md`](./TEAM_MANUAL.md)
- 知识库：[`KNOWLEDGE_BASE.md`](./KNOWLEDGE_BASE.md)

## Repository structure (proposed)

- `TEAM_MANUAL.md` 团队手册（总纲）
- `KNOWLEDGE_BASE.md` 知识库入口（索引）
- `kb/` 知识库正文（按主题分类）
- `docs/` 交付物沉淀（brief/insight/growth/product/engineering/retro）
- `adr/` Architecture/Decision Records（可选）
